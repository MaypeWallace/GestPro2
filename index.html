Tutorial do Sistema

Introdução

Este sistema foi desenvolvido para registrar e analisar paradas de máquinas na fábrica. Ele permite acompanhar os motivos das paradas, tempos de inatividade e gerar relatórios detalhados.

Seleção de Máquina

Na tela inicial, selecione a máquina para a qual deseja registrar uma parada. Você verá todas as máquinas disponíveis em forma de cards.

Registrando uma Parada

  1. Preencha a data da parada (por padrão, é a data atual)
  2. Selecione o setor responsável
  3. Escolha o operador (filtrado por setor)
  4. Selecione o motivo da parada
  5. Escolha a ocorrência específica
  6. Informe a duração da parada em minutos
  7. Se necessário, adicione observações
  8. Clique em "Registrar Parada"

Dashboard

O dashboard apresenta gráficos e estatísticas sobre as paradas:

  • Distribuição por motivos
  • Tempo total de parada por máquina
  • Histórico de paradas por período
  • Tendências e comparativos

Exportação de Dados

Para exportar os dados de paradas:

  • Use o botão "Exportar PDF" para gerar um relatório em PDF
  • Use o botão "Exportar Excel" para gerar uma planilha detalhada
  • Para transferir dados entre máquinas, use o botão "Exportar/Importar Dados"

Área Administrativa

O painel administrativo permite configurar o sistema:

  • Gerenciar máquinas (adicionar, editar, remover)
  • Configurar motivos de paradas e seus códigos
  • Gerenciar setores e operadores
  • Definir atribuições de operadores às máquinas
  • Configurar preferências como unidade de tempo (minutos/horas)

Transferência de Dados

Para transferir dados entre máquinas:

  1. Na máquina de origem, clique em "Exportar Dados"
  2. Salve o arquivo JSON gerado
  3. Na máquina de destino, clique em "Importar Dados"
  4. Selecione o arquivo JSON salvo anteriormente
  5. Escolha se deseja substituir ou mesclar os dados existentes

Dicas e Boas Práticas

  • Registre as paradas imediatamente após ocorrerem para maior precisão
  • Use as observações para detalhar circunstâncias específicas
  • Exporte backups regularmente para evitar perda de dados
  • Consulte o dashboard regularmente para identificar tendências e problemas recorrentes

Tutorial do Painel Admin

Gerenciamento de Máquinas

Gerenciando Máquinas

Aqui você pode adicionar, editar e remover máquinas do sistema.

  • Digite o nome/código da máquina no campo e clique em "Adicionar"
  • Para editar, clique no ícone de lápis na linha da máquina
  • Para excluir, clique no ícone de lixeira (esta ação requer confirmação)
Motivos de Parada

Configurando Motivos de Parada

Defina os códigos e descrições dos motivos de parada.

  • Cada motivo tem um código numérico (ex: 10) e uma descrição (ex: "Ajustes")
  • Os códigos padrão incluem: 10 (Ajustes), 20 (Enroscos), 30 (Falhas), etc.
  • Você pode alterar os códigos existentes ou criar novos conforme necessário
Ocorrências Específicas

Gerenciando Ocorrências

Configure as ocorrências específicas para cada motivo de parada.

  • Selecione o motivo, defina um código e uma descrição para cada ocorrência
  • Exemplo: Para o motivo "10 - Ajustes", você pode ter ocorrências como "101 - Ajuste de pressão"
  • Use o filtro para visualizar apenas as ocorrências de um motivo específico
Gerenciamento de Setores

Configurando Setores

Adicione e gerencie os setores da empresa.

  • Defina setores como "Manutenção", "Produção", "PCP", etc.
  • Os setores são utilizados para categorizar registros de parada
  • Se estiver usando vinculação por máquina, os setores ainda estarão disponíveis para seleção em cada registro
Gerenciamento de Operadores

Configurando Operadores

Adicione e gerencie os operadores da fábrica.

  • Digite o nome completo do operador e clique em "Adicionar"
  • Os operadores serão associados a setores ou máquinas através das atribuições
  • Os nomes dos operadores são utilizados nos registros e relatórios
Atribuições

Atribuição de Operadores

Vincule operadores a máquinas ou setores.

  • Selecione a máquina (ou setor, dependendo da configuração) e o operador
  • Clique em "Associar" para criar a vinculação
  • Esta função controla quais operadores aparecem disponíveis para cada máquina
  • Use o filtro para visualizar as atribuições por máquina ou setor
Configurações

Configurações do Sistema

Personalize o funcionamento do sistema conforme suas necessidades.

  • Unidade de Tempo: Alterne entre minutos e horas para registros de parada
  • Vinculação de Operadores: Escolha se operadores são vinculados a máquinas ou setores
  • Senha de Administrador: Altere a senha de acesso ao painel
  • Backup e Restauração: Faça backup dos dados ou restaure a partir de um arquivo
